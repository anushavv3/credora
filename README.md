# SkillTrust

A verified skilled worker network that connects clients with background-checked and certified professionals.

## Problem It Solves
Finding a trustworthy skilled worker (plumber, electrician, carpenter) is hard. There's no real verification, just unreliable ratings.

## Tech Stack
- Frontend: React, TypeScript, Tailwind CSS
- Backend: Node.js, Express.js
- Database: PostgreSQL with Prisma
- AI: Claude API, Google Vision API
- Payments: Stripe
- Real Time: Socket.io
